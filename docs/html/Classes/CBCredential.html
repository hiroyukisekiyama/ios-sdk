CBCredential Class Reference

Inherits from NSObject
Declared in CBCredential.h

Overview

CBCredential は Web サービスにおいて必要となる認証情報を Keychain を利用し、管理します。 対象ドメイン、ユーザー名の組合せをキーに、パスワード、BASIC 認証情報、クライアント証明書の管理ができます。
