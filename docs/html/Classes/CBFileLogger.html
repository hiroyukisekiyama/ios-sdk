CBFileLogger Class Reference

Inherits from NSObject
Declared in CBLog.h

Overview

ファイルロガーです。

CBLog にセットすることにより、ログをファイルに出力します。ログファイルは “Documents/Logs” に保存され、指定されたサイズ、ファイル数で循環します。
